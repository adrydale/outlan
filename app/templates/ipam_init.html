Database Initialization

Welcome to Outlan IPAM

It looks like this is your first time running Outlan IPAM, or the database needs to be initialized.

What will happen:

  • Create the database structure with all necessary tables
  • Set up audit logging and snapshot functionality
  • Initialize the system for IP address management

After initialization, you'll be able to:

  • Create network blocks to organize your subnets
  • Add subnets with VLAN assignments
  • Track changes with automatic snapshots
  • View audit logs of all activities

Note: This will create a new database file. If you have existing data, please ensure you have a backup before proceeding.
